# 🏀 Scoreboard Widget (Beta)

A lightweight, privacy-focused Android widget for a chronological morning check-in of your favorite sports.

## 🚀 Features
- **Chronological Feed:** See yesterday's finals and today's upcoming games in one list.
- **WNBA Focused:** Full support for all 15 teams, including the Golden State Valkyries.
- **Multi-League:** NBA, MLB, NHL, and NFL supported.
- **Privacy First:** No ads, no accounts, and no tracking (only basic Firebase for crash reports).

## 📲 Installation
1. Download the latest APK from the [Releases](https://github.com/greenfields-63/Scoreboard-Widget-Beta/releases) page.
2. Open the file on your Android device.
3. If prompted, allow "Install from Unknown Sources."

## 🛠️ Tech Stack
- Android Studio / Kotlin
- Firebase Crashlytics (for beta monitoring)
- Remote Configuration via GitHub Gist
