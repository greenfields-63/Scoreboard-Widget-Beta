# Scoreboard-Widget-Beta
Beta for Android Scoreboard widget
========================================================================
SCOREBOARD WIDGET - BETA V1.0
========================================================================

Thank you for participating in the open beta for this scoreboard widget! 
This is a lightweight, ad-free, and privacy-first Android widget 
designed to give you a clean, chronological morning check-in for your 
favorite teams across the NFL, NBA, MLB, NHL, and WNBA.

------------------------------------------------------------------------
1. HOW TO INSTALL (Sideloading the APK)
------------------------------------------------------------------------
Because this beta is distributed directly and not through the Google Play 
Store, Android requires you to manually approve the installation:

Step 1: Download the "Scoreboard_Widget_v1.0_Beta.apk" file from this Github repository onto your Android device.
Step 2: Tap the downloaded file to install it. 
Step 3: If prompted by your system ("For your security, your phone is 
        not allowed to install unknown apps from this source"), tap 
        "Settings" on that prompt.
Step 4: Toggle on "Allow from this source" (this temporarily grants 
        permission to your browser or file manager to install the file).
Step 5: Go back and tap "Install."

------------------------------------------------------------------------
2. PRIVACY & PERMISSIONS
------------------------------------------------------------------------
This app is built with a privacy-first approach:

* ZERO Ads.
* NO Account Creation or Personal Data Collection.
* ANALYTICS: This beta version utilizes Firebase Analytics/Crashlytics. 
  This is used strictly to identify app crashes and performance issues 
  across different Android devices so they can be fixed. No personally 
  identifiable information is tracked.
* THE GIST: The app is configured to check a secure GitHub Gist 
  periodically. This allows the app to notify you if a critical 
  update is available or if the data feed endpoint has changed.
* PERMISSIONS: The only permission requested is standard Internet 
  access so the widget can fetch live game scores and check for 
  essential updates.
